# Yaratech - Architecture d'Intérieur & Aménagement sur Mesure

Yaratech est un portfolio professionnel d'architecture d'intérieur basé au Maroc, spécialisé dans la conception 3D, l'aménagement de bureaux et le design résidentiel haut de gamme.

## Fonctionnalités
- **Portfolio Immersif** : Présentation dynamique des réalisations.
- **Page Services Dédiée** : Détail complet des domaines d'expertise (Avelar Style).
- **Étude de Projet en Ligne** : Formulaire de devis interactif.
- **Design Responsive** : Optimisé pour mobile et tablette.
- **Performance** : Site statique ultra-léger et rapide.

## Technologies Utilisées
- HTML5 / CSS3 (Variables CSS, Flexbox, Grid)
- JavaScript (Gestion dynamique du portfolio)
- FontAwesome (Icônes)
- Google Fonts (Outfit, Playfair Display)

## Comment lancer le projet localement
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/VOTRE_UTILISATEUR/yaratech-portfolio.git
   ```
2. Ouvrez `index.html` dans votre navigateur.
